# Facade print specification

## Purpose

Most Gothic visual detail should be printed on outdoor vinyl film. This keeps the plywood structure simple and makes the design easier to reproduce.

## Print layers

Recommended printed elements:

- Stone wall blocks.
- Weathering and shadows.
- False floor separation bands.
- Stained-glass window art.
- Rose window art.
- Slate roof tiles.
- Decorative arches and tracery shadows.
- Optional labels or alignment markers for cutting real windows.

## Print material

- Outdoor vinyl film.
- UV-resistant ink.
- Optional matte protective laminate.
- Use separate panels for front, rear, left side, right side, and roof slopes.

## Application workflow

1. Build and paint the Stage 1 plywood structure.
2. Sand and clean surfaces.
3. Apply facade film.
4. Use printed cut lines as window templates.
5. Cut only selected windows.
6. Seal cut edges.
7. Install acrylic/polycarbonate from inside.
8. Add LED backlight where needed.

## Printed vs real windows

### Printed-only

Best for:

- Simplicity.
- Waterproofing.
- Areas with no need for lighting.

### Cut-out windows

Best for:

- Main visible side windows.
- Rose window.
- Areas with warm LED backlight.

## Scale

Templates should be 1:1 in millimeters.

Include:

- Trim line.
- Safe margin.
- Optional cut line.
- Screw/decoration reference points.
