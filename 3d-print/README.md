# 3D printed decorative modules

## Purpose

Use 3D printing for visible Gothic depth, while keeping the main plywood structure simple.

## Recommended materials

- PETG for easier outdoor prints.
- ASA for better UV/weather resistance.
- Avoid PLA for permanent outdoor use.

## Suggested modules

- Tower / bell turret.
- Spires.
- Cross.
- Rose window frame.
- Entrance arch.
- Columns.
- Buttresses.
- Erkers / bay-window modules.
- Roof finials.
- Decorative vent covers.

## Design rules

- Make modules replaceable.
- Prefer screw mounting from inside or hidden mounting tabs.
- Avoid thin fragile tips where possible.
- Add drain/vent gaps where parts can trap water.
- Split large parts into printable sections.
