# Baseline dimensions

> Verify all dimensions against the real charging station before cutting final panels.

## Recommended external footprint

| Parameter | Value |
|---|---:|
| Length | 1200 mm |
| Width | 1000 mm |
| Wall height | 650–700 mm |
| Roof ridge height | 1050–1150 mm |
| Visual height with tower | 1300–1500 mm |

## Recommended internal robot bay

| Parameter | Value |
|---|---:|
| Internal length | ~1080 mm |
| Internal width | ~860 mm |
| Clear robot height | 600–650 mm |
| Gate width | ~700 mm |
| Gate height | ~420 mm |

## Service/storage compartment

The upper service compartment is created by a removable ceiling above the robot bay.

Suggested clear height:

- Minimum: 250 mm
- Comfortable: 300–350 mm

Suggested storage zones:

- Rear shelf for documents and warranty papers.
- Side shelf for spare blades, screws, cleaning brush.
- Ventilated power supply area with cable gland.
- Small removable box for accessories.

## Gate sizing logic

The front gate should be practical, not visually huge.

Recommended:

- Width: robot width + safe side clearance.
- Height: robot height + safe vertical clearance.

A large Gothic arch can be printed or added as facade decor around the practical gate opening.

## False floors

The building can visually show 2–3 floors without adding real internal floors:

1. **Main floor**: actual robot bay.
2. **Second false floor**: front gable and printed facade band with rose window.
3. **Third false floor**: small tower or bell turret above roof ridge.

## Roof access

Recommended options:

### Lift-off roof

Simpler and more weatherproof.

- Held by magnets, latches, or locating pins.
- Easy full access.
- Must be heavy or latched enough for wind.

### Hinged roof

More convenient for frequent access.

- Use stainless hinges.
- Add a stay/prop.
- Add rubber seal or drip edge.
- Ensure the roof can open without hitting the tower.
