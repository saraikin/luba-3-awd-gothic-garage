# Baseline dimensions

> Verify all dimensions against the real charging station before cutting final panels.

## Technical reference dimensions

Current reference dimensions are based on available LUBA 2 AWD / charging station data and should be verified against the real Luba 3 AWD installation.

| Item | Length | Width | Height |
|---|---:|---:|---:|
| LUBA 2 AWD robot | 690 mm | 513 mm | 273 mm |
| Charging base / station MTL23CHS0001 | 610 mm | 570 mm | 350 mm |
| Minimum shelter internal size | 900 mm | 600 mm | 500 mm |

## Docking / undocking clearance

The garage does **not** need internal maneuvering space. The robot docks and undocks along one fixed trajectory.

Critical placement requirement:

- Provide about **1500 mm of straight, level, obstacle-free path in front of the charging station**.
- This straight path is outside the garage footprint.
- Do not place columns, thresholds, stones, curbs, steps, or decorative elements in this approach path.
- The entrance should guide the robot along the same trajectory as the charging station.

## Recommended external footprint

| Parameter | Value |
|---|---:|
| Length | 1000–1200 mm |
| Width | 800–1000 mm |
| Wall height | 600–700 mm |
| Roof ridge height | 1050–1150 mm |
| Visual height with tower | 1300–1500 mm |

A smaller footprint is possible because internal maneuvering space is not required. The remaining extra volume is mainly for wall thickness, cable routing, service access, airflow, and tolerance.

## Recommended internal robot bay

| Parameter | Value |
|---|---:|
| Internal length | 900–1080 mm |
| Internal width | 600–860 mm |
| Clear robot height | 500–650 mm |
| Gate width | ~700 mm |
| Gate height | ~420 mm |

Minimum viable internal bay:

- 900 × 600 × 500 mm

Comfortable build target:

- ~1000–1080 × 700–860 × 600 mm

## Service/storage compartment

The upper service compartment is created by a removable ceiling above the robot bay.

Suggested clear height:

- Minimum: 250 mm
- Comfortable: 300–350 mm

Suggested storage zones:

- Rear shelf for documents and warranty papers.
- Side shelf for spare blades, screws, cleaning brush.
- Ventilated power supply area with cable gland.
- Small removable box for accessories.

## Gate sizing logic

The front gate should be practical, not visually huge.

Recommended:

- Width: robot width + safe side clearance.
- Height: robot height + safe vertical clearance.

A large Gothic arch can be printed or added as facade decor around the practical gate opening.

## False floors

The building can visually show 2–3 floors without adding real internal floors:

1. **Main floor**: actual robot bay.
2. **Second false floor**: front gable and printed facade band with rose window.
3. **Third false floor**: small tower or bell turret above roof ridge.

## Roof access

Recommended options:

### Lift-off roof

Simpler and more weatherproof.

- Held by magnets, latches, or locating pins.
- Easy full access.
- Must be heavy or latched enough for wind.

### Hinged roof

More convenient for frequent access.

- Use stainless hinges.
- Add a stay/prop.
- Add rubber seal or drip edge.
- Ensure the roof can open without hitting the tower.
