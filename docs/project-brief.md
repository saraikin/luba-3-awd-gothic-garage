# Project brief

## Goal

Create an open DIY project that helps Luba 3 AWD owners build a compact weather-protective robot garage with the appearance of a small Gothic church.

The project should be practical first and decorative second. The mower must be protected and able to dock reliably before any complex visual details are added.

## Target audience

- Mammotion Luba 3 AWD owners.
- DIY builders with basic woodworking tools.
- Users who want a decorative garden object rather than a simple technical shelter.

## Build philosophy

1. **Functional core first**: a reliable box and roof that protect the robot and charging station.
2. **Decorative skin second**: printed film provides most stone, slate, and stained-glass visuals.
3. **3D decor third**: printed elements add depth without making the plywood structure too complex.

## Key requirements

- Minimal footprint around the charging station.
- Separate robot bay and service/storage area.
- Removable or hinged roof.
- Dedicated place for power supply, spare parts, and documentation.
- Ventilated electrical compartment.
- No water traps around plywood joints.
- No high threshold at the robot entrance.
- Design should tolerate small build inaccuracies.

## Architectural concept

The visual style is a miniature Gothic chapel:

- Low practical gate for the mower.
- Tall false front gable for visual height.
- 2–3 false floors expressed through facade graphics and decorative trim.
- Central tower or bell turret above the roof ridge.
- Side erkers / bay-window modules as optional non-structural decor.
- Rose window above entrance.
- Narrow stained-glass side windows.
- Warm internal and window backlighting.

## Important design decision

Use a simple rectangular structural box with battens, then add Gothic visual layers on top. Do not make the main structure dependent on complex decorative geometry.
