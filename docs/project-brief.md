# Project brief

## Goal

Create an open DIY project that helps Luba 3 AWD owners build a compact weather-protective robot garage with the appearance of a small Gothic church.

The project should be practical first and decorative second. The mower must be protected and able to dock reliably before any complex visual details are added.

## Target audience

- Mammotion Luba 3 AWD owners.
- DIY builders with basic woodworking tools.
- Users who want a decorative garden object rather than a simple technical shelter.

## Core architecture

The accepted architecture has two main modules:

1. **Lower functional module** — the actual working garage.
2. **Upper architectural module** — a hinged Gothic roof/fronton/tower assembly added later.

The lower module can function alone. The upper module is an upgrade that adds the church silhouette and opens on rear hinges to access the service shelf.

## Build philosophy

1. **Functional core first**: a reliable lower box that protects the robot and charging station.
2. **Flat printed facade second**: printed vinyl makes each module look finished without requiring relief or complex woodworking.
3. **Hinged architecture third**: the gable roof and frontons are a separate upper module, not part of the lower wall.
4. **Towers and physical decor last**: tower boxes, corner towers, 3D parts, cornices, and LEDs are optional upgrades.

## Incremental build rule

Every stage must be a valid final stopping point:

- Stage 1 already works as a garage.
- Stage 2 already looks visually finished at the lower level.
- Stage 3 adds the roof/fronton silhouette without changing the lower module.
- Stage 4 completes the visual shell of the upper module with printed film.
- Later stages add towers and physical decor without destroying or discarding previous work.

Avoid destructive steps:

- Do not build temporary towers that must be thrown away later.
- Do not require replacing the lower facade when the upper module is added.
- Do not make physical relief mandatory before the printed facade version is complete.

## Key requirements

- Minimal footprint around the charging station.
- No internal maneuvering area is required: the robot docks and undocks along one fixed trajectory.
- Keep about 1500 mm of straight, level, obstacle-free approach path in front of the charging station.
- Concrete slabs can be used as the ground base.
- No plywood floor is required if the robot stands directly on the slabs.
- The lower module has a horizontal plywood ceiling that also acts as a service shelf.
- The service shelf stores power supplies, spare parts, documentation, and future LED components.
- The upper architectural module should open on hinges for access.
- Dedicated place for power supply, spare parts, and documentation.
- Ventilated electrical/service compartment.
- No water traps around plywood joints.
- No high threshold at the robot entrance.
- Design should tolerate small build inaccuracies.

## Architectural concept

The visual style is a miniature Gothic chapel/cathedral:

- Low practical gate for the mower.
- False floors expressed by rows of printed false windows, not by internal rooms.
- Gable roof and frontons belong to the hinged upper module.
- Central tower or bell turret belongs to the upper module.
- Optional corner tower modules also belong to the upper module.
- Rose window and stained-glass windows can be printed first and only later replaced with physical details if desired.
- Warm LED backlighting is optional and should be planned but not required for the first build.

## Important design decision

Use a simple rectangular lower functional box with battens and a flat ceiling/service shelf. Add Gothic visual layers and architecture as separate modules on top. Do not make the lower structure dependent on complex decorative geometry.
