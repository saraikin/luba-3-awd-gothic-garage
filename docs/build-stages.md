# Build stages

## Stage 1 — Basic protective garage

Purpose: protect the robot and charging station as quickly and reliably as possible.

Build:

- Rectangular plywood box.
- Timber batten frame.
- Floor/base panel.
- Side walls.
- Rear wall.
- Front wall with practical mower opening.
- Removable or hinged roof.
- Internal ceiling panel separating robot bay from service/storage compartment.
- Weatherproof coating.

Do not spend time on Gothic windows or decor at this stage.

Recommended checks:

- Robot enters and exits without touching the gate.
- Charging station remains aligned.
- Cable does not interfere with robot movement.
- Roof can be removed/opened easily.
- Water cannot run directly into the robot bay.

## Stage 2 — Facade, windows, and lighting

Purpose: turn the technical shelter into a Gothic-style object.

Add:

- Outdoor printed vinyl film.
- Stone wall texture.
- Slate roof texture.
- Printed or cut stained-glass windows.
- Optional acrylic panels behind real cut-outs.
- Warm LED backlight behind windows.
- Ventilation grilles hidden inside decorative patterns.

Recommended approach:

1. Apply printed facade film as a positioning template.
2. Cut only the windows that truly need light.
3. Keep some windows printed-only for better waterproofing.
4. Seal all cut plywood edges before installing acrylic.

## Stage 3 — 3D decor and false floors

Purpose: add the Gothic church character without complicating the core box.

Add:

- Front false gable.
- 2nd false floor visual layer.
- 3rd false floor / small tower.
- Spires.
- Cross.
- Rose window frame.
- Arches around gate.
- Columns / buttresses.
- Erkers / bay-window modules.
- Roof finials.

3D printing recommendation:

- Prefer PETG or ASA for outdoor parts.
- Avoid PLA for long-term outdoor exposure.
- Design decor as replaceable modules screwed from inside or attached to hidden inserts.

## Stage 4 — Refinement and community variants

Later project additions:

- Alternative dimensions for other Luba models.
- CNC-ready version.
- Laser-cut facade version.
- Parametric OpenSCAD/Fusion model.
- Multiple visual styles: Gothic chapel, castle gate, cottage, modern shed.
